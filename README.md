# Ground-Control-Station
This is a Ground Control Station (GCS). I made it for my team for the TEKNOFEST 2026 Unmanned Surface Vehicle (USV) Competition. With this program, we can see live data from our Unmanned Aerial Vehicle (UAV) and our Unmanned Surface Vehicle (USV). We can also control both vehicles from one screen.
I made this Ground Control Station (GCS) — in Turkish, Yer Kontrol İstasyonu (YKİ) — for my team. We use it in the TEKNOFEST 2026 Unmanned Surface Vehicle (USV) Competition. I did not use a program that already exists. I wrote my own program in Python, because in our mission we use two vehicles together: an Unmanned Aerial Vehicle (UAV) and an Unmanned Surface Vehicle (USV).
The GCS connects to both vehicles at the same time. It uses the MAVLink protocol. It shows live data from each vehicle: position (latitude and longitude), speed, roll, pitch, heading, battery level and flight mode.
On the main screen there is a live map. The map shows where both vehicles are. There is also an artificial horizon (HUD), a data panel for each vehicle, and buttons like arm/disarm and mode change. We can choose which vehicle we want to control. We can also see the logs of both vehicles at the same time, on one screen.
The program also does what the competition rules ask for. It can draw the graphs from the rules: real speed and speed setpoint, real heading and heading setpoint, and the force commands for each thruster. It can also save the data to a CSV file, because the competition wants this. And for the last task of the competition, the two vehicles can talk to each other through the GCS: the color that the UAV finds is sent to the USV automatically. We can turn this on and off with a button.
